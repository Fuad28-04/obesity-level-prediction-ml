# Obesity Level Prediction Using Machine Learning

## Overview
A multi-class classification project that predicts obesity risk levels based on lifestyle and health attributes such as diet, physical activity, BMI, and transportation habits.

## Problem Statement
Predict 7 different obesity levels from personal health and lifestyle data to assist in early health risk detection.

## Dataset Features
- Gender, Age, Height, Weight
- Family history with overweight
- Dietary habits (high-caloric food, vegetable consumption)
- Physical activity frequency
- Daily water consumption
- Mode of transportation

## Models Used
- Random Forest
- Support Vector Machine (SVM)
- Logistic Regression
- Decision Tree

## Technologies
- Python
- Scikit-learn
- Pandas, NumPy
- Matplotlib, Seaborn
